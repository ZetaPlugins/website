# ZetaPlugins

We are a group of developers who create plugins for Minecraft.

## Our Principles

Open Source

We believe in open-source software because it gives you the freedom to make it your own. Whether you want to add new features or adapt it for a specific use case, the code is open and ready for you to build on.

Quality

We build our plugins with a strong focus on performance, security, and user experience. Every detail matters, and we're committed to making sure everything runs smoothly and reliably.

Customizability

We know every server is different, so our plugins are built to be flexible. You can easily adjust settings and features to fit your setup and make things work just the way you want.
